Struttura a due colonne con un div contenitore a display:grid; grid-template-column: 1fr 1fr; da tenere a centro pagina. Cliccando sul pulsante in alto si aggiunge/toglie una classe che aggiunge/toglie margin:auto; al div contenitore che permette di centrare il contenitore. Vedi file app.js per l'azione.

Two columns layout with a div container set with display:grid; grid-template-column: 1fr 1fr; to set at center of the page. Clicking on the button above will toggle a class with margin:auto; to the container div. See app.js file for the action.
